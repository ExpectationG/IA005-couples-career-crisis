## 项目简介

### 项目背景

- 一位是陷入职业焦虑，在读的武汉大二学生，一位是在长沙已毕业三年，渴望转行的自由职业者。两人已相爱相杀相守半年载，今日不来秀恩爱，是来为彼此做职业生涯规划。我们不仅为自己做规划，也思考伴侣如何在职业规划上 *一起* 进步。
